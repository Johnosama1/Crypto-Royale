# Crypto Royale - بوت تيليجرام للروليت

## نظرة عامة
بوت تيليجرام يتيح للمستخدمين إنشاء وإدارة سحوبات "روليت" على تيليجرام. من تطوير **JOHN OSAMA**.

## التقنيات المستخدمة
- **لغة البرمجة:** Python 3.12
- **المكتبة الرئيسية:** `python-telegram-bot` v20.7
- **التخزين:** ملف JSON محلي (`data.json`)
- **البرمجة غير المتزامنة:** asyncio + aiofiles

## هيكل المشروع
- `bot.py` - الملف الرئيسي لتشغيل البوت
- `data.json` - قاعدة بيانات المستخدمين والقنوات
- `count_users.py` - سكريبت لعرض إحصائيات المستخدمين
- `requirements.txt` - المكتبات المطلوبة

## الإعداد
- **التوكن:** يُقرأ من المتغير البيئي `TELEGRAM_BOT_TOKEN` (مع قيمة احتياطية)
- **تشغيل البوت:** `python3 bot.py`

## الأوامر الرئيسية
- `/start` - القائمة الرئيسية
- `/count` - إحصائيات المستخدمين
- `/topusers` - أكثر المستخدمين تفاعلاً
- `/userinfo` - معلومات عن مستخدم

## الميزات
- تسجيل المستخدمين تلقائياً
- ربط قنوات تيليجرام بالمستخدمين
- إنشاء روليت مع شروط الاشتراك في قنوات
- إحصائيات تفصيلية للمستخدمين
